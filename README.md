## Power consumption

- Power supply voltage: 9V
- LDO Regulator ([AMS1117-3.3](https://datasheet.lcsc.com/lcsc/2204131745_Shenzhen-Fuman-Elec-AMS1117-3-3V_C173386.pdf)): 3.3V
- Power Dissipation (quiescent): 0.034W
